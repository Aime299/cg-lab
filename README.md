实验二：旋转与变换（三角形）
实验目标
深入理解 3D 空间中的坐标变换流程（MVP 变换），独立推导并用代码实现模型变换、视图变换和投影变换矩阵。

三角形顶点坐标
v0: (2.0, 0.0, -2.0)

v1: (0.0, 2.0, -2.0)

v2: (-2.0, 0.0, -2.0)

核心功能
1. 模型变换 (Model Transformation)
绕 Z 轴旋转的变换矩阵

2. 视图变换 (View Transformation)
将相机平移到世界坐标系原点

3. 投影变换 (Projection Transformation)
透视投影矩阵，分为两步：

透视平截头体挤压为正交长方体

正交投影至规范立方体 [-1, 1]³

交互操作
按 A 键：三角形顺时针旋转

按 D 键：三角形逆时针旋转

按 ESC 键：退出程序
